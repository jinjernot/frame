User Guide
  1. Download the CAP XML File: Content Aggregator for Products.
  2. Load the File into the Required Form: Upload the CAP file into your desired report generation tool. Depending on your analysis needs, you can generate different types of reports, including:
    • Product Only: Return images containing only the product
    • Annotated Image: Return images with annotations that highlight key features, issues, or areas of interest.
    • Image URL: Generate a report listing image URLs, identifying potential broken links or inaccessible images.
  3. Analyze and Extract Insights: Utilize the tool's features to process the image data, filter relevant information, and display the required content in a clear and organized manner. This step facilitates troubleshooting and issue spotting by highlighting key areas of interest and anomalies.
